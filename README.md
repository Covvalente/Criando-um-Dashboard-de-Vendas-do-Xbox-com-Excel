# Xbox Game Pass Subscription Analysis

Este projeto apresenta uma análise de dados de assinaturas do **Xbox Game Pass** utilizando o Microsoft Excel como ferramenta principal para organização, tratamento e visualização dos dados.

A proposta do projeto é explorar informações relacionadas aos assinantes, identificar padrões de faturamento e analisar o impacto de serviços adicionais na receita total.

---

## Objetivo do Projeto

O projeto foi desenvolvido com foco na análise de indicadores relacionados às assinaturas do Xbox Game Pass, buscando responder perguntas como:

* Qual o faturamento total das assinaturas?
* Qual o impacto da renovação automática na receita?
* Quanto os serviços adicionais contribuem para o faturamento?
* Como os diferentes tipos de plano influenciam os resultados?

---

## Estrutura do Projeto

O arquivo está organizado em quatro abas principais:

### Assets

A aba contém os elementos visuais utilizados no dashboard, como:

* Paleta de cores
* Ícones
* Referências visuais
* Identidade gráfica do projeto

---

### Bases

Contém a base principal de dados utilizada nas análises.

Os dados incluem:

* ID do assinante
* Nome
* Tipo de plano
* Data de início da assinatura
* Tipo de assinatura
* Renovação automática
* Valor da assinatura
* Serviços adicionais contratados
* Cupons aplicados
* Valor total por cliente

---

### Calculos

A aba reúne os cálculos e análises realizadas com tabelas dinâmicas e indicadores.

Entre as principais análises estão:

* Faturamento total dos planos
* Receita por tipo de assinatura
* Impacto da renovação automática
* Receita gerada pelos serviços adicionais
* Consolidação dos indicadores utilizados no dashboard

---

### Dashboard

Apresenta a visualização final dos dados de forma interativa.

O dashboard contém:

* Indicadores de faturamento
* Resumo das assinaturas
* Receita dos serviços adicionais
* Segmentações para filtragem dos dados
* Visualização consolidada das métricas do projeto

---

## Principais Insights

A análise dos dados permite observar que:

* O plano Ultimate apresenta maior valor agregado entre os planos disponíveis.
* Os serviços adicionais possuem participação relevante no faturamento total.
* A renovação automática impacta diretamente a receita recorrente.
* Diferentes modelos de assinatura influenciam o comportamento dos clientes.

## Como Utilizar

1. Abra o arquivo Excel do projeto.
2. Acesse a aba `Dashboard`.
3. Utilize os filtros e segmentações disponíveis para explorar os dados.
4. Consulte as abas `Bases` e `Calculos` para visualizar os dados e indicadores utilizados.
